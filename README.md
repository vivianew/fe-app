## Quick Start
Node.js 8 or higher is required.

Run in browser, try responsive view! :)

  Get the repo:

```bash
$ git clone https://github.com/vivianew/fe-app.git
```

  Install dependencies:

####  npm
```bash
$ npm install
```

OR

####  Yarn
```bash
$ yarn
```

  Start the server:

```bash
$ npm start
```

  Start tests:

```bash
$ npm t
```
